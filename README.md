# MovieServerManager
A PHP Movie Server Web Manager.

## Required
1. Aria2
2. php*-curl
3. php*-ssh2

## Environment(.env)
```
SERVER_HOST=localhost
SERVER_PORT=22
SERVER_USERNAME=pi
SERVER_PASSWORD=rsapberrypi
SERVER_ARIA=http://localhost:6800/jsonrpc
```
